# bun-modules-ts-static-server

## Usage

Write code in `src/web` directory

## Instalation

To install:

```bash
git clone https://github.com/Voldemortas/bun-modules-ts-static-server .
git remote remove origin
```

## Running

Dev mode:

```bash
bun run dev
```
